Travel Snapshot is an **interactive web application** that allows users to explore countries, view popular attractions, check current weather, and manage their favorite places around the world. Built with modern web technologies, it provides an intuitive and visually appealing map-based interface for global exploration.

---

## Features

- **Interactive World Map:** Click on countries to view detailed information.
- **Country Details:** Displays capital, region, population, and currency.
- **Weather Integration:** Shows current temperature and wind speed for the selected country.
- **Attractions List:** View major attractions with descriptions and image galleries.
- **Favorites Management:** Logged-in users can add attractions to their favorites.
- **Responsive Design:** Works well on both desktop and mobile devices.
- **Wiki Links:** Quick access to Wikipedia pages for deeper information on attractions.

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript, [Leaflet.js](https://leafletjs.com/) for maps
- **Backend:** Python, FastAPI
- **Database:** PostgreSQL (hosted on Render)
- **Authentication:** JWT token-based login system
- **APIs:** Weather, country info, and attractions served from the backend
